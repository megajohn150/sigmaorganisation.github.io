This is a guide on how to post your website for everyone to see using git and github

  • create a local git repository (git init)
  • add files to it (git add .)
  • save changes (git commit -m "your message")
  • create a online repository on github (create new, insert name without capital letters
    and add to it ".github.io", and choose if you want it to be public or private)
  • link your local repository to your online one (git remote add origin "url to your online repository")
  • upload the local repository to the online one (git push -u origin main)
  • the name of your online repository is an link to your website!
